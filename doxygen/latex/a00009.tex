\subsection{Detailed Description}
Helps with serializing access to a database connection. 

mutexes are only useful when threading mode is set to \char`\"{}\-Serialized\char`\"{}.

This mutex class can be used with \href{http://en.cppreference.com/w/cpp/thread/lock_guard}{\tt std\-::lock\-\_\-guard} as it meets the \href{http://en.cppreference.com/w/cpp/concept/BasicLockable}{\tt Basic\-Lockable} requirements.
