\subsection*{What is S\-Q\-Lite\-X\-X}

A C++ wrapper for sqlite3 that uses features in C++14.

You will need a compiler that supports C++14. The Travis-\/\-C\-I Y\-A\-M\-L file shows some of the supported compilers.
